
>>> $CLI connections create --help
Create a connection.
  
  Creates a new connection
  
  Creates a new connection to an external data source. It allows users to
  specify connection details and configurations for interaction with the
  external server.

Usage:
  databricks connections create [flags]

Flags:
      --comment string   User-provided free-form text description.
  -h, --help             help for create
      --json JSON        either inline JSON string or @path/to/file.json with request body (default JSON (0 bytes))
      --read-only        If the connection is read only.

Global Flags:
      --debug            enable debug logging
  -o, --output type      output type: text or json (default text)
  -p, --profile string   ~/.databrickscfg profile
  -t, --target string    bundle target to use (if applicable)
