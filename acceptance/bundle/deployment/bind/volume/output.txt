=== Create a pre-defined volume:
>>> [CLI] volumes create main [SCHEMA_NAME] [VOLUME_NAME] MANAGED
{
  "catalog_name": "main",
  "full_name": "main.[SCHEMA_NAME].[VOLUME_NAME]",
  "schema_name": "[SCHEMA_NAME]",
  "volume_type": "MANAGED"
}
